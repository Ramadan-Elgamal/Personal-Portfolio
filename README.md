# R3B Portfolio

A minimal, pixel-perfect dev portfolio and blog to showcase my work as a Software Engineer.

## About

Hi, I'm Ramadan Elgamal (R3B) - an Egyptian software developer and Computer Science graduate with three years of experience specializing in frontend development. Currently enrolled in the ITI Open Source 9-Month program.

## Stack

- Next.js 16
- Tailwind CSS v4
- shadcn/ui

## Featured

- Clean & modern design
- Light/Dark themes
- SEO optimized (JSON-LD schema, sitemap, robots)
- Installable as PWA

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
